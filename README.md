# environments
This repository contains environments for the Pythia platform.
Available environments are built with *Debian stretch* packages.

## Available environments

- Bash (Bash 4.4)

- Busybox

- C (GCC 4.3)

- Go (Golang 1.7)

- Java (OpenJDK 8)

- Lua (Lua 5.3)

- Mono (Mono 4.6.2.7)

- Python (Python 3.5.3-1)
